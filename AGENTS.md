# AGENTS.md (isfna)

Если ты агент и работаешь в этом репозитории:

1. Сначала прочитай `isfna.bootstrap.json`.
2. Затем прочитай `skills/master-init/SKILL.md`.
3. Не добавляй «всё подряд» — выбирай минимально полезный набор.
4. Любое утверждение о качестве подтверждай eval-результатами.
5. Для развития используй `skills/self-learning-loop/SKILL.md`.
