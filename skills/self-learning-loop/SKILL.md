# Self Learning Loop

Используй после первичной инициализации навыков.

## Цель

Эволюционировать набор навыков без деградации качества.

## Цикл

1. **Выбери гипотезу улучшения**
   - Что именно улучшится? (например, strict score, скорость, честность отчётов)
2. **Внеси минимальное изменение**
   - 1 гипотеза = 1 изменение.
3. **Прогони eval**
   - Сравни с baseline.
4. **Прими решение**
   - Если лучше или не хуже без новых рисков — оставить.
   - Если хуже — откатить.
5. **Задокументируй**
   - Обнови changelog и baseline (если осознанно улучшили критерии).

## Правила

- Никаких «массовых апдейтов» без промежуточной валидации.
- Если нет evidence — нет claim.
- Если метрики спорные, предпочитай стабильность.
